# password-strength-analyzer
🔐 Password Strength Analyzer
📌 About the Project
This is a real-time Password Strength Analyzer web application that evaluates password security based on length, complexity, entropy, and estimated crack time. It also provides suggestions to improve weak passwords.
🚀 Features
Real-time password strength check
Strength meter (Weak / Medium / Strong)
Entropy calculation
Estimated crack time (offline attack simulation)
Security improvement suggestions
Clean cybersecurity-themed UI
🛠️ Built With
HTML
CSS
JavaScript
🧠 How It Works
User enters a password
JavaScript analyzes:
length
uppercase/lowercase letters
numbers
special characters
A score is generated to determine strength
System calculates entropy and crack time
Suggestions are shown for weak passwords
